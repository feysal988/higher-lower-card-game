# Higher / Lower Card Game

## How to Run
Run Main.java from IntelliJ.

## Game Rules
A standard 52 card deck is used.
Ace is treated as the highest card.
The player guesses whether the next card will be higher or lower.

## Design Decisions
Enums are used for suits and ranks.
Game rules are separated from game flow.
Deck handles card creation and shuffling.

## Improvements
GUI interface
Additional game modes
Jokers
Unit tests

## Added Features
Streak-based scoring system that rewards consecutive correct guesses.
Two Joker cards added to the deck which reset streaks and allow the player to continue.
